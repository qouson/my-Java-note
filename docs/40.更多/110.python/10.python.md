## 常见数据采集框架

- Requests：
  - http库
- BeautifulSoup：
  - html和xml解析库
- Scrapy：
  - 强大的爬虫框架，用于从网上抓取结构化数据
- Selenium：
  - 自动化测试工具，可以用来做网页数据采集
- PyQuery：
  - 类似于JQuery的Python版本
- Pandas：
  - 强大的数据分析能力
- Urllib：
  - python自带的网络请求库
- Lxml：
  - 高效的xml和html解析库，提供xpath支持，比BeautifulSoup更强大，适用于更复杂的数据解析任务。
