## CPU

- vmstat
![20210317101634](https://cdn.jsdelivr.net/gh/qouson/my-pic-bed/pic/20210317101634.png)

## 网络IO

- ifstat
  - 默认本地没有，下载ifstat
    - wget <http://gael.roualland.free.fr/ifstat/ifstat-1.1.tar.gz>
    - tar xzvf ifstat-1.1.tar.gz
    - cd ifstat-1.1
    - ./configure
    - make
    - make install
  - 查看网络IO
    - ifstat 1
      - 各个网卡的in、out
      - 观察网络负载情况
      - 程序网络读写是否正常
      - 程序网络I/O优化
      - 增加网络I/O带宽
